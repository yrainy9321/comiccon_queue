# 微信静默登录架构说明

## 📐 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                     微信小程序客户端                          │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  ┌──────────────┐                                           │
│  │  app.js      │ ← 应用启动时自动执行                       │
│  │              │                                            │
│  │ onLaunch()   │                                            │
│  │     ↓        │                                            │
│  │ 检查缓存     │ → 有缓存：直接使用                         │
│  │     ↓        │                                            │
│  │ wx.login()   │ → 无缓存：获取code                         │
│  │     ↓        │                                            │
│  │ 发送请求     │ → POST /api/wechat/login                   │
│  └──────────────┘                                            │
│                                                               │
│  ┌──────────────┐                                           │
│  │ globalData   │ ← 全局数据存储                             │
│  │ - openid     │                                            │
│  │ - token      │                                            │
│  │ - userInfo   │                                            │
│  └──────────────┘                                            │
│                                                               │
│  ┌──────────────┐                                           │
│  │ Storage      │ ← 本地持久化                               │
│  │ userInfo     │                                            │
│  └──────────────┘                                            │
│                                                               │
└─────────────────────────────────────────────────────────────┘
                              ↓ HTTPS
┌─────────────────────────────────────────────────────────────┐
│                      后端服务器 (Express)                     │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ POST /api/wechat/login                               │   │
│  │                                                       │   │
│  │ 1. 接收 code                                          │   │
│  │ 2. 调用微信API换取openid                              │   │
│  │ 3. 生成token                                          │   │
│  │ 4. 保存用户信息                                       │   │
│  │ 5. 返回结果                                           │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                               │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ GET /api/wechat/user                                 │   │
│  │                                                       │   │
│  │ 1. 验证token                                          │   │
│  │ 2. 返回用户信息                                       │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                               │
│  ┌──────────────┐                                           │
│  │ data/        │ ← 文件存储                                 │
│  │ wechat_users │                                            │
│  │ .json        │                                            │
│  └──────────────┘                                            │
│                                                               │
└─────────────────────────────────────────────────────────────┘
                              ↓ HTTPS
┌─────────────────────────────────────────────────────────────┐
│                    微信服务器 API                             │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  https://api.weixin.qq.com/sns/jscode2session               │
│                                                               │
│  输入: appid + secret + code                                 │
│  输出: openid + session_key                                  │
│                                                               │
└─────────────────────────────────────────────────────────────┘
```

## 🔐 安全机制

### 1. Token验证
```
客户端请求 → 携带token → 后端验证 → 通过/拒绝
```

### 2. Code一次性
```
wx.login() → code (5分钟有效，只能用一次)
```

### 3. 密钥保护
```
.env文件 → 不提交到Git → 环境变量读取
```

## 🚀 性能优化

### 1. 缓存策略
- **一级缓存**：globalData（内存，最快）
- **二级缓存**：StorageSync（磁盘，持久）
- **避免重复**：有缓存不调用wx.login()

### 2. 异步加载
- 登录过程不阻塞页面渲染
- Promise方式等待登录完成
- 超时保护（5秒）

### 3. 错误处理
- 网络异常降级
- 友好的用户提示
- 日志记录便于调试

## 📊 状态管理

### 用户状态机

```
[未登录] 
   ↓ wx.login()
[登录中] 
   ↓ 成功
[已登录] ←→ [Token过期] → [重新登录]
   ↓ 失败
[登录失败] → [重试/提示]
```

### 页面状态

```javascript
// 页面中检查用户状态
onLoad() {
  // 1. 等待登录完成
  app.getUserInfo()
    .then(userInfo => {
      // 2. 使用openid
      this.setData({ openid: userInfo.openid });
    })
    .catch(err => {
      // 3. 处理错误
      console.error('登录失败', err);
    });
}
```
